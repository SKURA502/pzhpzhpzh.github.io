本博客的第一篇文章

Posted on 2019-02-21

一点感想

很早之前就想开一个博客了，这次总算把它搞完啦，很感谢学长们的帮助，没有他们的帮助我可能就搭不出来这个博客了呢。

为什么要开这个博客，意义是什么呢？自己想了很久。。。虽然感觉自己今年就有可能退役了，但是感觉这个博客在我的日后的生活中也可以起到很大的作用（就像是写一些记录什么的）。感觉信息竞赛还是一个很靠积累的过程，经常有做过的题目过了一段时间就忘记了，感觉得不到很多的提升。所以就应该写博客把思路记录下来，能便于翻阅，在后期能给我帮上很大的忙。但毕竟自己还是一个刚学oi半年的蒟蒻，和很多大佬完全不能比，自己的做题思路可能比较的蠢笨，希望机房的大佬们不要嘲笑，经常给我提出一些意见呢。

当然我的博客里面可能也不全是做题记录呀，可能也会把自己给自己写的话记录下来，毕竟我是一个自制力没有那么强的人，很可能没有恒心去做好一个事情，所以就应该给自己写一些话鞭策自己，让自己走上正确的轨道呢。

又是一学期呢，pzh还是得加油呀，但是现在的pzh有了自己的方向，知道怎么努力了呢。
以上是我脑子一热敲出来的一段话呢，但也确实是我想说出来的，感谢您从百忙之中抽出时间来看这个呢。
