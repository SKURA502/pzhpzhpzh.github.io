[NOI2014]动物园

题目描述

求出一个num数组，输出所有num数组的乘积。

num数组即为一个串，相同前缀和后缀的个数。

输入输出格式

输入格式

第1行仅包含一个正整数nn ，表示测试数据的组数。
随后n行，每行描述一组测试数据。每组测试数据仅含有一个字符串s，s的定义详见题目描述。数据保证s 中仅含小写字母。输入文件中不会包含多余的空行，行末不会存在多余的空格。

输出格式

包含 n 行，每行描述一组测试数据的答案，答案的顺序应与输入数据的顺序保持一致。对于每组测试数据，仅需要输出一个整数，表示这组测试数据的答案对 1,000,000,007 取模的结果。输出文件中不应包含多余的空行。

解题思路

KMP思路还是很清奇的，没有搞懂这种题目完全不会做的说。。

但是大多数Kmp的题目都是围绕着神奇的nxt数组做文章，因此对nxt数组需要很好的掌握。(nxt:最长的相同前缀后缀的长度)。

说了这么多，来看题。题目中说要求一个串前缀后缀相同的个数。我们来看以下这个字符串。

​ a b c d 。 。 。 a b c d

大家可能觉得这个比较蠢，我们抽象的理解其为一个前后缀相等长度为4的字符串。如果我们除去abcd外，要再找到一个再相等的前后缀，此时会有以下几种情况：

1 . a = d （那么此时的nxt[4]==1）

2 . ab = cd (那么此时的nxt[3]==1&&nxt[4]==2)

我们可以从中看出，如果要找到此时串有较小的相同前后缀，我们只需要判断最长前缀中的最后一位判断是否有nxt值即可，那么我们用nxt不断往前跳，能跳到一次，那么就表示原串其能多一个满足条件的前后缀。然而只能拿五十
