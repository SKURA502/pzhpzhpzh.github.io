CF351E Jeff and Permutation

最先开始的时候对这道题无从下手，感觉不知道怎么做 果然是太菜了。听了学长讲之后才理解。

对于可以改变任何数的正负，那么它原始的值的正负显然是没有任何意义的，那么肯定与它的绝对值有关系。这里我们把绝对值记为h[N].

对于一个位置i，我们在[1,i-1]和[i+1,n]两段区间中考虑i的值改为负数给这两段区间造成的影响。

[1,i-1] 的情况（令a属于该区间）：

当h[a]<h[i]时，将i改为负数，那么h[a]>h[i],即改变逆序对的个数，而h[a]>h[i]时，将i改为负数的话逆序对个数并没有影响。那么只需要统计h[a]<h[i]的情况就好了

[i+1,n]的情况（令b属于该区间）：

当h[b]<h[i]时，很显然h[i]改为负数的时候h[b]>h[i],那么逆序对的个数很显然会改变，而h[a]>h[i]时，将i改为负数没有任何影响。所以这种情况下也统计h[b]<h[i]的情况。

综上所述，只有h[x]<h[i]时，逆序对数量才会改变，所以只需要求两区间分别改变的个数，ans加上其中小的值就行了

CF1110E Magic Stones

这是一道很难想到的思路题呢，根据题意中的条件，在c[ ]数组中每一个值都可以改为周围两个数之和减去中间这个值得到这个位置的新值，也就是

​ c [ i ] = c [ i + 1 ] + c [ i - 1 ] - c [ i ]

对于变换后的式子我们进行求当前的差分，会发现一些神奇的东西

​ c [ i + 1 ] - c [ i ] = c [ i + 1 ] - c [ i + 1 ] - c [ i - 1 ] + c [ i ] = c [ i ] - c [ i - 1 ]

​ c [ i ] - c [ i - 1 ] = c [ i + 1 ] + c [ i - 1 ] - c [ i ] - c [ i - 1 ] = c [ i + 1 ] - c [ i ]

两个差分数组只不过左右换了个位置而已，那么我们只需要比较两个差分数组是否相同就行了。
